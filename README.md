## ⚙️ Installation

### 1️⃣ Create Virtual Environment

```powershell
python -m venv venv
venv\Scripts\activate
```

### 2️⃣ Install Dependencies

```powershell
pip install -r requirements.txt
```

*or*

```powershell
pip install streamlit agno groq python-dotenv
```

### 3️⃣ Run App

```powershell
streamlit run ui.py
```
